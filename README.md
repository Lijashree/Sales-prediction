# 📊 Retail Sales Prediction & Analytics

## 📌 Overview

This project focuses on analyzing retail sales data and building machine learning models to predict future sales trends. It includes data preprocessing, feature selection, and ensemble-based forecasting techniques.

---

## 🚀 Features

* Data cleaning and preprocessing
* Feature selection using Granger Causality
* Ensemble machine learning models
* Sales forecasting and predictions
* Feature importance analysis

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* CatBoost
* Jupyter Notebook

---

## 📂 Project Structure

```
notebooks/        → Model building and analysis  
data/             → Dataset files  
results/          → Output predictions and analysis  
```

---

## 📊 Dataset

The dataset used is a retail supermarket sales dataset containing:

* Product categories
* Sales data
* Time-based features

---

## ⚙️ How to Run

1. Clone the repository:

```
git clone <your-repo-link>
```

2. Install dependencies:

```

```

3. Run notebooks:

```
jupyter notebook
```
